# 改版說明

## 概要
- 新增「市場聚焦」卡片，可切換目前預設的分析區域（台灣／東南亞／日本／美國），並同步更新畫面上顯示的選擇，使得 AI 比較思考更貼近實際市場。
- 將「競品比較矩陣」的欄位依「核心體驗」「市場/行銷」「資源能力/策略彈性」三大區塊重新分組，添加顏色標示與描述文字，提升立刻辨識能力。
- 在 `comparison-table` 下方加入資料來源說明卡片（附 tooltip 說明），列出核心體驗、市場行銷、資源能力三大區塊所依據的資料類型（如內部文件、競品官網、社群監測、AI 分析結果等）。
- 新增「競品公開資訊」說明卡片，會列出官網與產品頁、App/Platform 評分，以及新聞/媒體報導等欄位，並標注對應的資料來源提示。

## 新增功能
- `market-focus-card`：顯示目前市場區域並附下拉選單，方便使用者在分析前先定義聚焦地區，未來可以延伸至 API 篩選。
- `comparisonSourcesData` 陣列＋`renderComparisonSources()`：用來渲染來源說明卡片，顯示各區塊的取材類型與補充文字。
- `renderComparisonTable()` 改為分組輸出，在每組前插入色塊 header，並保持原有的分析欄位；新增單獨欄位 `comparison-sources` 用於放置來源說明。

## 資料與更新建議
- 若未來需要讓每筆比較紀錄都帶來源，可讓 AI 回傳的 `analysis` object 加上 `source` 欄位，前端在單元格下方顯示該欄位內容。
- 「市場聚焦」可接後端 API 以自動切換社群趨勢圖與 alert 的地區區分，或新增自訂市場清單。

## 測試
- 目前僅透過瀏覽器互動手動檢視 DOM 是否正確渲染，不含自動化測試。
